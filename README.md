# msh

Le projet msh consiste a developper un mini shell en langage C. L’objectif principal est de reproduire le comportement de base d’un interpreteur de commandes : lire une commande saisie par l’utilisateur, l’analyser, puis l’executer. Le shell doit aussi rester actif apres l’execution d’une commande afin de permettre a l’utilisateur d’en saisir une nouvelle.
